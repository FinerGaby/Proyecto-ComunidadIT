# Proyecto Final Comunidad IT
APP Mesa de ayuda ( CRUD ) 

  - Registro de usuario / Login
  - Crear Posts / Editar / Borrar / Comentarios
  - Perfiles

# Tecnologias usadas

  - Javascript
  - Handlebars
  - Node Js
  - Express
  - MongoDB
  - Morgan

### Installation
Instalar MongoDB y Ejecutarlo
Instalar dependencias usadas npm install

```sh
$ npm install
$ cd src
$ npm start
```
